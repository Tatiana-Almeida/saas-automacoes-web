# Guia do Usuário — Onboarding

Bem-vindo! Este guia rápido ajuda você a começar.

1. Cadastro: acesse a página de registro (endpoint `POST /api/v1/auth/register/`).
2. Confirme seu e-mail (se configurado) e faça login.
3. Crie um tenant (se aplicável) ou associe-se a uma organização.
4. Configure integrações (WhatsApp, SMS, Email) na área de integrações.
5. Envie sua primeira mensagem ou crie um workflow.

Suporte: use o formulário na landing page ou `POST /api/v1/support/tickets/` para abrir um ticket.
